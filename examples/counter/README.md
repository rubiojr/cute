# Cute Counter Example

A minimal desktop counter app built with Cute, the Shoes-inspired
UI toolkit for Rugo.

## Prerequisites

- Qt6 development libraries (`apt install qt6-base-dev` on Debian/Ubuntu)
- Go 1.22+
- miqt: fetched automatically on first build

## Build and run

```bash
rugo build main.rugo -o counter && ./counter
```
